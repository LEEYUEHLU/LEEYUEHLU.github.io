在CococsCreator設定測試環境

  1. 將"Test"資料夾放在專案的資料夾底下，如下呈現：(ProjectFolder / Test)。

  2. 將"package.json"放在專案的資料夾底下，如下呈現：(ProjectFolder / package.json)。

  3. 打開"Command Line"並且導覽到專案資料夾底下。

  4. 在"Command Line"中輸入"npm install"，並且按下"Enter"運行。

  5. 接著輸入"npm test" ,然後會在專案視窗中看到"mochawesome-report"的資料夾。

  6. 點選資料夾檔案中的mochawesome.html"，來瀏覽報告。

在CococsCreator設定 DI

  1. 使用 npm 安裝 Inversify。

  2. 引入inversify模組 : import { Container, ContainerModule, interfaces } from "inversify";

  3. 產生一個container : const myContainer = new Container();

  4. 用inversify的模組來定義物件相依條件 :

    let DTBind = new ContainerModule(
    (bind: interfaces.Bind, unbind: interfaces.Unbind) => {
    bind(TestDI).toSelf();});


  5. 使用load()來載入定義好的模組 : myContainer.load(DTBind )
