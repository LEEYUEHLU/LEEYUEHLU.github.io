在CococsCreator設定 DI

  1. 打開"Command Line"並且導覽到專案資料夾底下。

  2. 使用git clone此資料 https://github.com/LEEYUEHLU/PluginForCocosCreator.git

  3. 在專案新增inversify.config.ts

  4. 引入inversify模組 : import { Container, ContainerModule, interfaces } from "inversify";

  5. 產生一個container : const myContainer = new Container();

  6. 用inversify的模組來定義物件相依條件 :

    let DTBind = new ContainerModule(
    (bind: interfaces.Bind, unbind: interfaces.Unbind) => {
    bind(TestDI).toSelf();});


  7. 使用load()來載入定義好的模組 : myContainer.load(DTBind )
返回主頁
