在CococsCreator設定 DI

  1. 使用 npm 安裝 Inversify。

  2. 引入inversify模組 : import { Container, ContainerModule, interfaces } from "inversify";

  3. 產生一個container : const myContainer = new Container();

  4. 用inversify的模組來定義物件相依條件 :

    let DTBind = new ContainerModule(
    (bind: interfaces.Bind, unbind: interfaces.Unbind) => {
    bind(TestDI).toSelf();});


  5. 使用load()來載入定義好的模組 : myContainer.load(DTBind )
